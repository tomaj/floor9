01 What is Cortex

From git push to production. One brain.

Cortex is our AI-first DevSecOps platform. It started with a frustration: outside work, deploying an app takes seconds - GitHub, Vercel, push, done. At work, the same thing takes a week of manual pipelines, hand-written manifests, and chasing approvals.

Cortex closes that gap. 11 modules, one ecosystem - from CI templates to AI ops agents. Everything automated, secured, and connected.
02 The 11 modules
→ CI Templates - 200+ lines of .gitlab-ci.yml, written from scratch, in every project. Now: 2 include lines = entire pipeline. Build, test, scan, deploy. 22 security checks baked in.
→ Feature Previews - every branch gets its own environment automatically. Push a branch, get a live URL. Test in full isolation - no more "it works on my machine." Something we simply couldn't do before.
→ Docker Images - hardened, signed, auto-patched base images. Outside Cortex: 60-163 CVEs. Through Cortex: 1.
→ Helm Charts + GitOps - 3 typed charts replace hand-written k8s manifests. ArgoCD auto-syncs every deploy. Rollback, canary, promotion - all built in.
→ Security Policies - vulnerabilities and banned licenses blocked before merge. SAST, secret detection, container scan, SonarQube, OWASP - all automatic.
→ Minions - AI ops agents (Bedrock + Claude). Self-healing brokers, capacity prediction, auto-runbooks. 3 AM incident? The AI handles it.
→ Renovate Bot, E2E Utils, Infra & Cache, Wiki, Switchboard - auto-dependency updates, shared test mocks, build cache (jobs in 20s not 3 min), centralized docs, Backstage developer portal.
